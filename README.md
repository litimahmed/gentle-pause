## Prerequisites

Before running the project, make sure you have the following installed:

- **Node.js** (v14 or higher)
- **npm** (v6 or higher)
- **Git**

---

## Installation

1. **Clone the repository**

   ```bash
   git clone https://gitlab.com/anis_amara/toorrii_equipe
   cd home-toorrii
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```
